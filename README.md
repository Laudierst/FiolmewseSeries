## 🔥 Instalação e execução

1. Faça um clone desse repositório;
2. Entre na pasta `cd Potfolio`;
5. Rode `yarn init -y` e depois `yarn install` para rodar a aplicação;
6. Rode `yarn start`;
7. Acesse a URL `http://localhost:3000`;
